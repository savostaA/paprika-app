# 🌶️ Паприка — Кейтеринг

Мобильное приложение для кейтеринговой компании «Паприка» (Северодвинск / Архангельск).

## 🚀 Быстрый старт

### 1. Запуск сервера (бэкенд)
```bash
cd server
npm install
npm run dev
```
Сервер запустится на `http://localhost:3001`

### 2. Запуск клиента (фронтенд)
```bash
cd client
npm install
npm run dev
```
Клиент запустится на `http://localhost:5173`

### 3. Сборка APK
```bash
# Требования: Java JDK 17+, Android SDK
chmod +x build-apk.sh
./build-apk.sh
```

## 📱 Функциональность

### Клиент
- Каталог блюд с фильтрами по категориям и поиском
- Детальная карточка блюда (фото, описание, ингредиенты, КБЖУ, вес)
- Корзина с изменением количества
- Оформление заказа с выбором типа мероприятия
- Отслеживание заказов по номеру телефона

### Админ-панель
- Дашборд с статистикой (заказы, выручка)
- Управление меню (добавление/редактирование/удаление блюд)
- Загрузка фото через drag & drop
- Управление заказами (смена статусов)

## 🛠️ Технологии
- **Frontend**: React 18 + Vite
- **Backend**: Express.js + SQLite
- **APK**: Capacitor 7
- **Стиль**: Vanilla CSS (тёмная тема)
